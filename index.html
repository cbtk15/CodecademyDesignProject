Buttons & Links

Nav Links

Main navigation links should be consistent as much as possible to enusre the user can identify the right affordances and interact with the links in a faster way. One way we can do this is by using a identifier which draws a underline from the left to the right when the user hovers over the nav link.

Hover Example

The animation alone is achieved using the following Css:


            /* Keyframes for the Underscore Animation */

            @keyframes nav-width {
              0%{
                width: 0;
              }
              100% {
                width: 28.422rem;
              }
            }

            /* Link and Pseudo Element Rulesets */

            .ch-nav-a-6:link{
              text-decoration: none;
             }
                 
            .ch-nav-a-6::before{
                 content: "";
                 width: 0;
                 height: 3.25px;
                 visibility: hidden;
                 background-color: hsl(195deg 93% 35%);
                 position: absolute;
                 bottom:0px;
                 left: 0;
                 transition: all 0.3s ease-in-out;
                 transition-property: width, 
                 visibility, background-color;
               }
             
               .ch-nav-a-6:hover::before{
                 visibility: visible;
                 width: 100%;
               }
             
               .ch-nav-a-6:active::before{
                 background-color: hsl(195deg 83% 43%);
                 transition: 0.1s ease-in-out;
               }
